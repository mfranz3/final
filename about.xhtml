Data Collection-General Methodology

  • Reviewed the health section (exact one reviewed specified under each below) of each news site
  • Took note of anything that pertains to women's health within the United States (US)
  • Looked for a "key" word in the title, including "women," "pregnancy," "abortion," "abortion-pill," "ivf," etc.
  • An id, place, key[word], lat, long, date, author, and source were recorded for each item
  • For place:
    • If it referenced a specific location, I used that.
    • If it spoke about a general health topic, but referenced the location of one main study, I used that.
    • If the article did not specify a specific place, I used "DC" as the location, as it is the US capital.
    • If the "place" is a state, the coordinates were inputted as the state's capital city.
    • If the location (ie, the NIH) has multiple locations, the coordinates of the headquarters were used.

1. New York Times Collection

  • Reviewed the title of all articles under the "health" section, back to November 1, 2023 (~6 months)

Mapping and Visualization Methodology

  • I mapped the data using ArcGIS Online software, importing the data from .csv files.
    • The CSV files contained the following:
      • id: identification number.
      • place: location of where the research is happening or where the reporting is referencing.
      • lat: latitude.
      • long: longitude.
      • key: key word(s) from the title.
      • author: author of the article.
      • title: title of the article.
      • url: url of the article.
  • The maps displayed on the website are downloads of the maps created using ArcGIS online.
  • Next, I transformed all of the CSV files into XML files, using Regex to markup items so that each item from the CSV file was its own element.
    • Example to mark up <keys>:
      • find: ([key]),
      • replace:<key type="\1">\1</key>,
  • Using XSLT, I created a transformation to make a Chicago Style source list/bibliography for each XML page (ie, for each data base).

Issues With Collection/Methodology

  • The collection of data (ie, articles) was done by hand, so it is subject to human error.
  • Additionally, place was collected by hand, so it may also be subject to human error.
  • However, the general goal of this project is to get a sense of number of locations/reports done on women's health, and to allow users to find those studies easily.
  • Although the maps display the number of articles, I did not look to compare the number of articles published, so I did not control for number of total number of articles published in general by that news outlets. Neither did I compare to number of articles published specifically regarding men's health.
